 # Number Guessing Game

A simple web game where the player or the computer tries to guess a number.
This project is built using HTML, CSS, and JavaScript and is deployed using GitHub Pages .


 Game Modes:-
 
1) Player Guesses

- The computer randomly selects a number.
- The player tries to guess it before the timer runs out.
- Hints like Bigger or Smaller help guide the player.

2) Computer Guesses
- The player thinks of a number.
- The computer tries to guess it.
- The player tells the computer if the number is bigger, smaller, or correct.


 Difficulty Levels:-
 Easy → smaller range
 Medium → moderate range
 Hard → larger range and more challenge


 Features:
* Two game modes
* Difficulty selection
* Timer challenge
* Confetti animation when winning
* Responsive design (works on phone and desktop)
